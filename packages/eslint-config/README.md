## Installation

Install `@wardes/eslint-config`:
``` bash
npm i -D eslint @wardes/eslint-config
```

``` bash
yarn add -D eslint @wardes/eslint-config
```

``` bash
pnpm add -D eslint @wardes/eslint-config
```

Add the following to your eslint config:

```json
{
  "extends": [
    "@wardes/eslint-config"
  ]
}
```
